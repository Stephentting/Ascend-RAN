# Ascend-RAN
基于昇腾架构的新一代智能无线基带处理平台

**Ascend-RAN** 旨在打破传统RAN架构的限制，结合华为昇腾AI处理器的强大算力，构建一个面向未来的AI-Native基带处理平台。我们致力于实现通信与AI的深度融合，为6G及大规模智能网络提供自主可控的技术底座。

## 算子算法开源状态

### 无线通信算子算法

我们已经成功开源了Ascend-RAN平台的核心算子算法！
这些无线算子算法经过精心设计和优化，充分利用昇腾NPU的异构计算能力，旨在提升无线基带处理的性能和能效。目前开源的包括：

*   **信道估计算子**：实现高效精准的信道状态估计。
*   **信号检测算子**：例如优化的ZF（迫零）检测算子，相比通用计算，性能提升显著。
*   **LDPC 解码算子**：大幅优化了解码效率，降低了计算耗时，提升了吞吐量。
*   **QAM 调制/解调算子**：实现高效的信号调制与解调功能。
*   **FFT/IFFT算法**：通过重构为矩阵运算，充分利用NPU矩阵计算单元，大幅加速傅里叶变换。
*   **矩阵求逆算子**：通过迭代近似方法，适配NPU硬件特性，降低MIMO等场景的复杂度。

### 在线YOLO检测 (Online YOLO Detection)开源状态
*   **目标**：集成NPU加速的实时AI推理能力，服务于AI-RAN的智能应用。
*   **内容**：提供针对昇腾NPU优化的YOLO系列模型推理代码及在线检测示例，并与通信链路结合。

这些无线算子算法的开源是Ascend-RAN项目的重要一步，它们为构建更灵活、更智能的无线通信系统奠定了基础。

## 项目愿景

我们坚信，通过通信与计算的深度融合，能够突破现有“卡脖子”困境。Ascend-RAN将持续迭代，提供一套可控、可演进的AI-RAN底座，支撑6G与大规模智能网络的演进。

1.  **语义通信 (Semantic Communication) 模块**
